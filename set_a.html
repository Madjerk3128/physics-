PART – A    (10 × 2 = 20 Marks)

Answer All Questions – All Questions are Compulsory

1. Define atomic packing factor. State the packing efficiency values for SC, BCC, and FCC structures.
2. What is Czochralski method? State the materials commonly grown by this method.
3. Define Young's modulus. State the formula for maximum depression in a simply supported beam loaded at its centre.
4. What is twisting couple? Write its expression for a solid cylindrical rod of length l, radius r, and rigidity modulus n.
5. Define simple harmonic motion. Write its differential equation.
6. What is damped oscillation? Define damping coefficient.
7. Define numerical aperture of an optical fibre. Write its formula in terms of refractive indices of core and cladding.
8. What is acceptance angle of an optical fibre? How is it related to numerical aperture?
9. What is zero-point energy? Write its value for a particle in a 1D box of length L.
10. State Heisenberg's Uncertainty Principle. Write the mathematical expression for position-momentum uncertainty.

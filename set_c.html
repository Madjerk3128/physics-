PART – A    (10 × 2 = 20 Marks)

Answer All Questions – All Questions are Compulsory

1. Define packing factor. Which crystal structure — SC, BCC, or FCC — has the highest packing efficiency and why?
2. What is Molecular Beam Epitaxy? State one device application for which it is primarily used.
3. What is moment of inertia of a beam cross-section? For a rectangular cross-section of breadth b and depth d, write the formula for I.
4. What is rigidity modulus? How does it differ from Young's modulus? State SI units of both.
5. State the condition for a motion to be simple harmonic. Give two examples of SHM.
6. Define Q-factor of a resonant system. What does a high Q-factor indicate?
7. Define numerical aperture. What is the maximum possible value of NA for a fibre? Justify.
8. What is attenuation in optical fibres? State its unit and formula.
9. State Heisenberg's energy-time uncertainty principle. What is its physical implication for excited state lifetimes?
10. What is degeneracy in quantum mechanics? Give the degeneracy number for the first excited state of a particle in a 3D cubic box.
